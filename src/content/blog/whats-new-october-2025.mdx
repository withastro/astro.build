## Tips & Tools

The latest community-built [utilities and integrations](/integrations/) to help you build with Astro. üõ†

- [üåü Starlight Links](https://github.com/HiDeoo/starlight-links) - Visual Studio Code extension providing IntelliSense for links in Markdown and MDX files for Starlight documentation projects.
- [üåü Mixedbread Search for Starlight](https://github.com/mixedbread-ai/mixedbread-starlight) - A Mixedbread integration for Starlight to replace the default Pagefind search.
- [üåü Starlight docsearch typesense](https://github.com/typesense/starlight-docsearch-typesense) - A Starlight plugin that adds DocSearch powered by Typesense for fast, typo-tolerant documentation search.
- [Astro Chatty GPT](https://github.com/unfolding-io/astro-chatty-gpt) - A simple chatbot api for your Astro site. Powered by Upstash search, gpt-5 and AI-sdk.
- [Astro ATProto OAuth Starter](https://tangled.org/@daffl.xyz/astro-atproto-starter) - A minimal Astro starter template demonstrating OAuth authentication with AT Protocol (ATProto), the decentralized social networking protocol used by Bluesky and other services.
