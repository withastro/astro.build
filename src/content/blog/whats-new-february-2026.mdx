## Tips & Tools

The latest community-built [utilities and integrations](/integrations/) to help you build with Astro. üõ†

- [AIS Logos Astro](https://npmx.dev/package/@foreview/ais-logos-astro) - An Astro integration for loading AI Safety organization logos.
- [Subfont](https://npmx.dev/package/@forthgoing/subfont) - An Astro integration that subsets your font files so you get significantly smaller font sizes and faster loading speeds.
- [Astro Particle Image Viewer](https://npmx.dev/package/@ivanalbizu/astro-particle-image-viewer) - An Astro component that displays images with a stunning WebGL particle animation effect using Three.js.
- [Astro Openring](https://npmx.dev/package/@jasikpark/astro-openring) - A custom Astro content loader that combines multiple RSS/Atom feeds into a single collection. Mimics the behavior of openring / openring-rs, a tool for generating webrings.
- [Astro Bun Adapter](https://npmx.dev/package/@wyattjoh/astro-bun-adapter) - Astro adapter for Bun with optimized static file serving and ISR (Incremental Static Regeneration).
- [Astro Cache](https://npmx.dev/package/@wyattjoh/astro-cache) - Astro integration providing disk-backed SWR (stale-while-revalidate) and memo caching utilities powered by `flat-cache` and `stale-while-revalidate-cache`.
- [Astro Skills](https://npmx.dev/package/astro-skills) - Load and serve Agent Skills from your Astro site via well-known URIs.
- üåü [Starlight Announcement](https://npmx.dev/package/starlight-announcement) - A plugin to add custom announcements to your Starlight docs.
- üåü [Starlight Telescope](https://npmx.dev/package/starlight-telescope) - Quickly navigate to any page in your Starlight docs with fuzzy search and keyboard-first navigation.
- [Astroscope Health](https://npmx.dev/package/@astroscope/health) - Kubernetes-style health probes integration for Astro.
- [Astro Clarity](https://npmx.dev/package/@bitfresh/astro-clarity) - Astro integration for Microsoft Clarity.
- [Entrolytics Astro SDK](https://npmx.dev/package/@entrolytics/astro-sdk) - Official Astro SDK for Entrolytics - first-party growth analytics for the edge.
- [Astro Maplibre-YAML](https://npmx.dev/package/@maplibre-yaml/astro) - Astro components for creating interactive maps and scrollytelling experiences using YAML configuration.
- [Astro ONREZA Adapter](https://npmx.dev/package/@onreza/adapter-astro) - Adapter for deploying Astro applications on the ONREZA platform with support for SSR, hybrid mode, prerendering, and ISR.
- [VocaSync Astro](https://npmx.dev/package/@vocasync/astro) - Turn your Astro blog posts into narrated audio with word-level synchronization.
- [Astro Font Loader](https://npmx.dev/package/astro-font-loader) - Loads fonts into astro build artifacts from custom fonts packages.
- [Astro ID Minifier](https://npmx.dev/package/astro-id-minifier) - An Astro plugin to minify Astro custom data attributes and reduce build size.
- [Astro Relative URLs](https://npmx.dev/package/astro-relative-urls) - This Astro integration transforms resource and page URLs from absolute to relative, making it so you can view the built website by simply opening its `index.html` in your browser.
- [Astro Routify](https://npmx.dev/package/astro-routify) - A high-performance API router for Astro using a Trie-based matcher.
- [Astro XMDX](https://npmx.dev/package/astro-xmdx) - An Astro integration for xmdx, a high-performance MDX compiler.
- [Astro Simple Critical CSS](https://npmx.dev/package/@gezellig/astro-simple-critical-css) - An Astro integration that generates critical CSS using beasties and inlines it into the HTML.
- [Astro Contrast](https://npmx.dev/package/@ivanalbizu/astro-contrast) - WCAG color contrast analyzer for Astro components. Checks your .astro files for color contrast issues at build time, dev time, or on demand via CLI.
- [Astro Junction](https://npmx.dev/package/@junctionjs/astro) - Astro v5+ integration for Junction. First-class View Transitions support.
- [Astro Primitives](https://npmx.dev/package/@minimall.io/astro-primitives) - Astro primitives inspired by Radix UI Themes.
- [Playform Favicon](https://npmx.dev/package/@playform/favicon) - An Astro integration that brings favicon generation utilities to your Astro project.
- [CTRK Astro](https://npmx.dev/package/@tex0l/ctrk-astro) - An Astro integration for Yamaha Y-Trac CTRK telemetry parser with Vue.js components and composables.
- [Astro Markdown Runtime](https://npmx.dev/package/@ze-moreno/astro-markdown-runtime) - Render Markdown at runtime in Astro using a custom AST pipeline that turns Markdown directives into real Astro components.
- [Zitadel Astro Auth](https://npmx.dev/package/@zitadel/astro-auth) - A comprehensive Auth.js integration for Astro applications with TypeScript support, framework-agnostic HTTP adapters, and role-based access control.
- [Astro Animations](https://npmx.dev/package/astro-animations) - Lightweight, zero-dependency, scroll-triggered animations and CSS scroll-linked effects for Astro.
- [Astro Linter](https://npmx.dev/package/astro-linter) - Astro integration that adds ESLint with pre-configured rules.
