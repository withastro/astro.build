## Latest releases

Minor releases introduce new features and improvements. Pre-releases introduce breaking changes, stabilize features, and provide improvements. Check out what's new!

- [Astro 5.18](https://github.com/withastro/astro/releases/tag/astro%405.18.0) - a new option to configure the maximum size of Astro Actions request bodies, and an `X-Forwarded-Proto` validation fix!
- [Astro 6.0-beta-17](https://github.com/withastro/astro/releases/tag/astro%406.0.0-beta.17) - the latest beta release of Astro v6 and sneak peek of the [v5 -> v6 upgrade guide draft](https://v6.docs.astro.build/en/guides/upgrade-to/v6/) with up-to-the-minute v6 docs, so you can feel free to try it!
