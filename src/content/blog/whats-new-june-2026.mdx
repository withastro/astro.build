The latest community-built [utilities and integrations](/integrations/) to help you build with Astro. 🛠

- [@anubis609/astroanimate-core](https://github.com/astroanimate/astroanimate#readme): Astro-native animation component library.
- [@arraypress/analytics-astro](https://github.com/arraypress/analytics-astro#readme): Multi-provider analytics for Astro — Plausible, Cloudflare, Umami, Fathom, Simple, GoatCounter, GA4, and custom.
- [@arraypress/reveal-astro](https://github.com/arraypress/reveal-astro#readme): View-transition-safe scroll-reveal for Astro — one IntersectionObserver adds a visible class as elements enter the viewport. Optional default animation CSS.
- [@casoon/astro-structured-data](https://github.com/casoon/astro-structured-data): An Astro integration to automatically generate structured data (JSON-LD) for articles, FAQs, products, recipes, videos, breadcrumbs, local businesses, events, and more.
- [@freshjuice/astro-webmcp](https://github.com/freshjuice-dev/astro-webmcp): Astro integration that exposes your site content via WebMCP for AI agents.
- [@hevmind/ask](https://github.com/hev/ask#readme): A heading-anchored, agentic search overlay for Astro docs sites.
- [@kckempf/astro-standard-site](https://github.com/kckempf/astro-standard-site#readme): Astro integration for standard.site - unified ATProto longform publishing.
- [@sjohansson/astro-components](https://github.com/sjohansson/astro-components#readme): A collection of modern, well-tested Astro components built with TypeScript, designed for Astro 5, 6, and 7 projects.
- [@stanniel/astro-pagefind](https://github.com/stanniel/astro-pagefind): Add Pagefind to your Astro projects.
- [@tinloof/astro-prefetch](https://github.com/tinloof/astro-kit#readme): Next.js-style prefetching for Astro's ClientRouter: in-memory cache, cursor-trajectory (proximity) prediction, and instant navigations.
- [@vskstudio/takt-astro](https://github.com/vskstudio/takt-astro#readme): Astro integration for Takt privacy-friendly analytics.
- [@xingwangzhe/satteri-mermaid](https://github.com/xingwangzhe/satteri-mermaid#readme): Satteri MDAST plugin for Mermaid diagram detection and transformation.
- [@xingwangzhe/satteri-photoswipe](https://github.com/xingwangzhe/satteri-photoswipe#readme): Satteri HAST plugin that wraps images with PhotoSwipe lightbox containers.
- [@xsynaptic/astro-build-logger](https://github.com/xsynaptic/astro-lab/tree/main/packages/astro-build-logger#readme): Astro integration that appends build duration, page count, and output size to a JSONL log file.
- [astro-bootstrap-icons](https://github.com/chanython/astro-bootstrap-icons#readme): Bootstrap Icons for Astro.
- [astro-convex-better-auth](https://github.com/Kaderimon/astro-convex-better-auth#readme): Astro plugin to have auth server-side.
- [astro-docs](https://github.com/shravanngoswamii/astro-docs#readme): A customizable documentation and book framework for Astro. Bring your own look.
- [astro-gamified-starter](https://github.com/oops-games-llc/astro-gamified-starter): Replace CAPTCHAs and visual puzzles with interactive, 3-second micro-games. Test the API and play the games [in the live sandbox](https://conversion-business-widgets.web.app/portal.html).
- [astro-password](https://github.com/unix/astro-password#readme): Password protection middleware for Astro.
- [astro-scute](https://github.com/thegu5/astro-scute): Simple and cute Standard.site integration for Astro.
- [astro-search](https://github.com/elstron/astro-search#readme): A search plugin for Astro that provides static search capabilities.
- [astro-sparkline](https://github.com/MarvinBregiosa/astro-sparkline#readme): Tiny inline-SVG sparkline component for Astro. Zero runtime, zero dependencies, server-rendered.
- [astro-webmcp](https://github.com/fabricioctelles/astro-webmcp): Astro integration that exposes your site content via WebMCP for AI agents.

### SEO/AEO

- [@arraypress/seo-astro](https://github.com/arraypress/seo-astro#readme): Idiomatic SEO component for Astro — title, canonical, Open Graph, Twitter, robots, hreflang, article meta + JSON-LD.
- [@rgglez/astro-seo](https://github.com/rgglez/astro-seo): Astro component for SEO meta tags, Open Graph, Twitter cards, hreflang links, robots directives, and JSON-LD.
- [@thenewdynamic/astro-seo](https://github.com/theNewDynamic/astro-seo): Astro integration for SEO meta tags and structured data.
- [astro-aeo-image](https://github.com/vsima/astro-aeo-image#readme): Astro image service that embeds your `<Image>` alt text (and optional description/keywords) as standards XMP into the optimized output files so your `dist/_astro` assets are self-describing for Google Images and AI answer engines.

### CMS/text editors

- [astro-block-cms](https://github.com/jamalapriadi/astro-block-cms): Reusable Astro integration and block renderer for CMS APIs with JSON blocks.

### Markdown, Text, and more

- [astroprint](https://github.com/atomiechen/astroprint#readme): Astro integration for print-ready, Markdown-first documents with paged preview and PDF export.

### Content loaders

- [@probablyduncan/astrena](https://github.com/probablyduncan/astrena#readme): A content loader that fetches blocks from an are.na channel.

### Dev toolbar apps

- [astro-show-route-template-components](https://github.com/danny-englander/astro-show-route-templates#readme): Dev-only Astro integration that injects HTML comments showing which route and template own each page and which .astro component produced each node.

### Starlight

- [@wave-rf/astro-themed-mermaid](https://github.com/Wave-RF/astro-themed-mermaid#readme): Build-time, theme-aware Mermaid diagrams for Astro / Starlight.
- [@wave-rf/starlight-llm-tools](https://github.com/Wave-RF/starlight-llm-tools#readme): Starlight plugin that adds LLM-friendly tooling to your docs site.

### Astro integrations by and for other tools

- [@arpsw/astro-cms](https://github.com/arpsw/astro-cms#readme): Astro integration for the ARP (Laravel) CMS — shared API client, i18n/path resolution, and config wiring for ARP CMS sites.
- [@awesome.me/astro-lit](https://github.com/shoelace-style/astro-lit): Use Lit components within Astro.
- [@brainwebuk/astro-payload-live-preview](https://github.com/BrainWeb/astro-payload-live-preview): Astro component for Payload CMS server-side Live Preview — refreshes your SSR routes when documents are saved in the admin panel.
- [@contentbit/astro](https://github.com/agonist/contentbit/tree/main/packages/astro): Astro renderer for Content Blocks: `.astro` components for rendering validated documents, with per-block component overrides.
- [@dropinblog/astro](https://github.com/DropInBlog/astro#readme): DropInBlog rendered content for Astro projects.
- [@freshjuice/zest-astro](https://github.com/freshjuice-dev/zest/tree/main/packages/zest-astro): Astro integration for Zest cookie consent toolkit (GDPR/CCPA)
- [@workos/authkit-astro](https://github.com/workos/authkit-astro#readme): Official WorkOS AuthKit SDK for Astro.
- [astro-unhead](https://github.com/Nickersoft/astro-unhead#readme): Unhead.js middleware for Astro.
- [meno-astro](https://github.com/meno-so/meno-astro): Meno-Astro: runtime helpers + the round-trippable .astro dialect (emit/parse) that lets Meno use Astro files as a source-of-truth format.
- [sitedrift](https://github.com/joeseverino/sitedrift) - Visual review tool for comparing Astro dev previews against live production Cloudflare deployments. Compare DEV and LIVE with Split, Overlay/Diff, SEO, response deltas, notes, and MCP.
