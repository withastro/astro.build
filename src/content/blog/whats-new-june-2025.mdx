## Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. üõ†

- [Back Link](https://events-3bg.pages.dev/library/BackLink/) - A component that lets you flip the direction of the view transition animation when using Astro‚Äôs client router.
- [Astro Validate Env](https://github.com/adamhl8/astro-validate-env) - A environment variable validation integration for Astro that gives you control over when/where to validate environment variables and works in all contexts.
- [Astro Medium loader](https://github.com/ykocaman/astro-medium-loader) - An Astro loader that fetches and parses Medium RSS feeds by username. Supports canonical links, optional caching, and flexible storage strategies.
- [Astro content loader](https://github.com/dnlzro/astro-content-loader) - Load `.astro` files with the Astro Content Layer API, just as you would other formats (e.g. Markdown, MDX).
- [unDocs](https://github.com/K0Rhq/un) - A documentation framework built with Astro where you can build your own docs and component playground where you control the layout, components, and design.
- [Stripe Astro loader](https://github.com/notrab/stripe-astro-loader) - Fetch data from the Stripe API and use it in Astro collections.
- [Astro Payload local loader](https://github.com/jliguori1312/payload-astro-template/tree/main/astro-payload-local-loader) - Load and cache data from a Payload CMS instance during your Astro build, using the local API.
- [Astro Markdown prerender](https://github.com/johnzanussi/astro-markdown-prerender) - An Astro integration that automatically sets `route.prerender` based on frontmatter properties in your Markdown files.
- [Astro Loader Hashnode](https://github.com/raisedadead/astro-loader-hashnode) - A powerful content loader for integrating Hashnode blog posts into your Astro website using the Content Layer API.
- [Astro TOC generator](https://github.com/bhargav-bkpatel/astro-toc-generator) - A table of contents generator for Astro.js that works specifically with Sanity CMS blog content.
- [Astro Swiper](https://github.com/pascal-brand38/astro-swiper) - Astro component for Swiper, dedicated to slider / carousel / photo swiper / slide.
- [Astro Medium loader](https://github.com/ykocaman/astro-medium-loader) - An Astro loader that fetches and parses Medium RSS feeds by username.
- [Astro llms.txt](https://github.com/4hse/astro-llms-txt) - An Astro integration to generate AI‚Äëfriendly documentation files.
- [Astro terminal themes](https://github.com/danlourenco/astro-terminal-themes) - Convert terminal theme configurations (Ghostty, Warp, iTerm2, etc.) into Tailwind CSS themes for your Astro website.
- [Astro Mermaid](https://github.com/joesaby/astro-mermaid) - An Astro integration for rendering Mermaid diagrams with automatic theme switching and client-side rendering.
- [Astro SEO plugin](https://github.com/bhargav-bkpatel/astro-seo-plugin) - A zero-config SEO plugin for Astro.
- [Astro Strapi loader](https://github.com/VirtusLab-Open-Source/astro-strapi-loader) - An Astro integration that enables easy data loading from Strapi Content API.

- [Is it really foss?](https://isitreallyfoss.com/) - Where Projects are Evaluated to see if they're as free and open source as advertised.
