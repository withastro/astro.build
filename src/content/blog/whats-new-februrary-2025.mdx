## Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. üõ†


- [Starwind UI](https://starwind.dev/) - A set of powerful, accessible components for your Astro projects styled with Tailwind CSS v4.
- [Zorn Player](https://studio.sudovanilla.org/) - A video player for Astro with several customizable features.
- [Accessible Astro](https://accessible-astro.incluud.dev/) - Everything you need to build accessible, performant, and beautiful websites with Astro.
- [`astro-obsidian-starlight-notes-template`](https://github.com/adityatelange/astro-obsidian-starlight-notes-template) - A Starlight starter template that uses your Obsidian notes for content.
- [`astro-teleport`](https://github.com/ok-98/astro-teleport) - A lightweight Astro component for rendering content in a different part of the DOM using a Web Component.
- [`astro-typesafe-routes`](https://github.com/feelixe/astro-typesafe-routes) - An Astro integration for typesafe URL generation and routing.
- [`storyblock-loader`](https://github.com/nonsponsored/storyblok-loader) - A community-built loader for Astro Content Layer that fetches content from Storyblok's API.
- [`astro-qiita-loader`](https://github.com/t0yohei/astro-qiita-loader) - A Qiita loader for Astro, allowing you to use Qiita posts as content collections.
- [Bit](https://github.com/bitbons-ai/bit) - A CLI tool for scaffolding Astro + PocketBase projects.
- [`static-only-collections`](https://inox-tools.fryuni.dev/content-utils#static-only-collections) - Specify content collections that will only be used on static pages to reduce code built server-side and improve page performance.
- [Starlight View Transitions Plugin](https://events-3bg.pages.dev/library/StarlightPlugin/) - Add cross-document, browser-native view transitions to your Starlight site with one line in your Astro config.

... and some built *with* Astro:

- [StudioCMS](https://studiocms.dev/) - A dedicated CMS for Astro DB, with a fresh beta 8 release.
- [Tract Stack](https://tractstack.org/) - A publishing platform and no-code Astro website builder.
- [GitHub Activity](https://github-activity.trueberryless.org/) - View the recent GitHub activity for any user in a beautiful list.
- [Mayo App](https://github.com/mayo-dayo/app) - A self-hostable, web-based audio streaming app.
