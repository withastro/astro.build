### Tips & Tools

The latest community-built utilities and integrations to help you build with Astro.

- [`astro-react-i18next`](https://github.com/jeremyxgo/astro-react-i18next) - Integrates i18next and react-i18next seamlessly into your Astro website to provide robust i18n support for React components.
- [Astro Resend](https://www.npmjs.com/package/@hbertoson/astro-resend) - Easily send email notifications for form submissions using Resend in an Astro project. 
- [DeWP](https://delucis.github.io/dewp/) - Use your WordPress data in Astro projects
- [`starlight-site-graph`](https://github.com/fevol/starlight-site-graph/) - A Starlight plugin adding a graph component to the site's right-sidebar.
- [Astro NPM Loader](https://www.npmjs.com/package/astro-npm-loader) - Load information about NPM packages into a content collection.
- [Local AstroDB Viewer](https://marketplace.visualstudio.com/items?itemName=YuriAlvesGuernsey.localastrodb) - View and interact with .db files directly in Visual Studio Code.
- [Astro Mail Obfuscation](https://www.npmjs.com/package/astro-mail-obfuscation) - Protect email addresses and phone numbers in your Astro app's source code from bots using XOR-based obfuscation.
- [`react-alpine-store`](https://www.npmjs.com/package/react-alpine-store?activeTab=readme) - Allows for syncing between reactive state across React and Alpine's $store. See more information about Alpine's $store here.
- [Expressive Code Color Chips](https://delucis.github.io/expressive-code-color-chips/) - A plugin to add color previews to your CSS code examples
- [Astro AO3 loader](https://github.com/FujoWebDev/fujocoded-plugins/tree/main/astro-ao3-loader) - A AO3 data loader for Astro using the experimental content layer and AO3.js to load data from AO3 to your Astro website.
