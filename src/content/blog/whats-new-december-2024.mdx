### Tips & Tools

The latest community-built utilities and integrations to help you build with Astro:


- [Domain Expansion](https://domainexpansion.gg/) - an Astro integration that adds support for incremental builds.
- [Sonda](https://sonda.dev/frameworks/astro) - visualize your Astro bundles
- [`astro-word-loader`](https://astro-word-loader.netlify.app/) - load `.docx` and `.doc` files into an Astro Content Collection
- [`astro-typed-links`](https://github.com/florian-lefebvre/astro-typed-links) - an Astro integration that automatically get typed links to your pages
- [`astro-globals`](https://github.com/gingerchew/astro-globals) - an Astro integration that stores site data in a global virtual module
- [`@terrahq/astro-core`](https://www.npmjs.com/package/@terrahq/astro-core) - a collection of core Astro components designed to streamline asset management and simplify form creation
- [Flickr loader](https://astro-loaders.lekoarts.de/) - a content layer loader for the Flickr API
- [`github-repos-astro-loader`](https://github.com/chenasraf/github-repos-astro-loader) - a content loader for Astro that fetches GitHub repositories and their README files
- [`starlight-plugin-show-latest-version`](https://starlight-plugin-show-latest-version.trueberryless.org/) - show the latest released version of your Starlight plugin in your plugin's Starlight documentation
- [Bag of Tricks: `<TurnSignal />`](https://events-3bg.pages.dev/library/TurnSignal/) - set individual, per-link view transitions and choose different animations for each navigation

... and some built *with* Astro:

- [RSS Beauty](https://rss.beauty/) - Transform RSS feeds into beautiful reading formats for a better reading experience.
- [Astro `glob()` Loader Config Generator](https://astro-glob-loader-config-generator.netlify.app/) - Need to check your content collections `loader` configuration? Use this web app to generate the correct settings when using Astro's built-in `glob()` loader.
- [OG image maker](https://ogimagemaker.com/) - A free OG image generator tool you can use to create beautiful Open Graph images for your blog posts, articles, and social media posts.
- [Two idiots, one keyboard](https://two-idiots-one-keyboard.pages.dev/) - an engineering team-bonding game inspired by ThePrimagen "two idiots one keyboard" video
