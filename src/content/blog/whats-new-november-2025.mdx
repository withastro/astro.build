## Tips & Tools

The latest community-built [utilities and integrations](/integrations/) to help you build with Astro. üõ†

- [Astro Web Vitals](https://github.com/casoon/astro-webvitals) - A lightweight Web Vitals monitoring component for Astro with debug overlay and analytics support.
- [Astro Capo](https://github.com/felixicaza/astro-capo) - Automatically sort your head with capo.js in Astro.
- [Astro i18nya](https://www.npmjs.com/package/astro-i18nya) - I18n support for Astro.
- [Astro AI Coauther](https://github.com/AltairaLabs/astro-ai-coauther) - A developer-first AI documentation assistant for Astro.
- [Astro Cloudflare Pages Static](https://github.com/mattlyons0/astro-cloudflare-pages-static) - An Astro integration for client-side dynamic routes with Cloudflare Pages.
- [Astro Directify](https://github.com/oamm/astro-directify) - Declarative server-side control-flow directives for Astro.
- [Astro WhatsApp](https://github.com/dfrios/astro-whatsapp) - An Astro component to show a WhatsApp fixed floating icon in your website.
- [Mona Kiosk](https://github.com/monakit/mona-kiosk) - An Astro integration for monetizing your content with Polar.sh. 
- [Zephyr Astro](https://github.com/ZephyrCloudIO/zephyr-packages/tree/main/libs/zephyr-astro-integration#readme) - An Astro integration for Zephyr Cloud deployment and analytics platform.
- [Astro Orbit](https://github.com/delta-incubator/astro-orbit) - A theme package for Astro projects featuring a design system, reusable components, and Tailwind CSS v4 integration. 
- [Astro Snapshot](https://github.com/twocaretcat/astro-snapshot) - An Astro integration for generating screenshots of your pages automatically at build time. 
- [Silk Astro](https://www.npmjs.com/package/@sylphx/silk-astro) - Astro integration for Silk - zero-runtime CSS-in-TypeScript with islands architecture and partial hydration support.
- [Scalar API Reference for Astro](https://github.com/scalar/scalar/tree/main/integrations/astro) - A Scalar API Reference Component for Astro.
- [EdgeOne Astro](https://www.npmjs.com/package/@edgeone/astro) - An adapter to deploy your Astro site to EdgeOne Pages.
- [Astro Auth](https://github.com/mridang/astro-auth) - An Astro integration for Auth.js that provides seamless authentication with multiple providers, session management, and UI primitives that feel natural in Astro. 
- [Astro Dynamic Assets](https://github.com/BearStudio/astro-dynamic-assets) - An Astro integration to help you build custom images like Open Graph Image, Instagram post, etc., using React.
- [Astro Trakt.tv Loader](https://github.com/LekoArts/astro-loaders/tree/main/packages/trakt) - A Trakt.tv content loader for Astro's content layer.
- [Astro HTML Minifier Next](https://github.com/jonasgeiler/astro-html-minifier-next) - An integration to automatically minify all HTML assets generated by Astro during the build process using the provided `html-minifier-next` options.
- [AstroJS Atom](https://github.com/flcdrg/astrojs-atom) - A library for generating Atom RSS feeds in Astro projects. 
- [Vitest Browser Astro](https://github.com/ascorbic/vitest-browser-astro/) - Test Astro components in real browsers with Vitest Browser Mode.

... and some built with Astro!

- [Git Views](https://github.com/remvze/gitviews) - A GitHub profile and repo view counter you can add to your website.
- [Release Image Generator](https://github.com/trueberryless-org/release-image-generator) - Generate random images with customizable patterns, colors, and text overlays.
