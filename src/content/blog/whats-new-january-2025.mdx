## Tips & Tools

The latest community-built utilities and integrations to help you build with Astro:

- [`@wix/astro`](https://www.npmjs.com/package/@wix/astro) - The Wix Astro integration provides session management for your Astro project.
- [Astro GitHub file loader](https://github.com/gingerchew/astro-github-file-loader) - Load files stored in a Github Repository into your Astro Content Layer.
- [Astro Integration Template](https://astro-integration-template.netlify.app/get-started/pick-a-template/) - Building an Astro integration? Just choose a template and get started!
- [`astro-ghostcms-loader`](https://github.com/MatthiesenXYZ/ghostcms-loader) -Use your GhostCMS content with the Astro Content Layer.
- [`@polar-sh/astro`](https://github.com/polarsource/polar-adapters/tree/main/packages/polar-astro) - Payments and Checkouts made simple with Astro.
- [Starlight Spell Checker](https://starlight-spell-checker.trueberryless.org/) - This plugin brings multilingual spell-checking, equality, readability, English usage, quotes checks, and much more to your Starlight projects.
- [`astro-transition-event-polyfill`](https://github.com/louisescher/astro-transition-event-polyfill) - An integration that adds a polyfill for the astro:page-load event when view transitions are not used.
- [Astro server todo app](https://github.com/devgauravjatt/astro5-server-todo-app) - A simple Todo List App built with Astro and Better Auth.
- [`astro-font` v1.0](https://github.com/rishi-raj-jain/astro-font) - Automatically optimize your Custom Fonts, Local Fonts, Fonts over any CDN and Google fonts for performance.
- [`@inox-tools/server-islands`](https://inox-tools.fryuni.dev/server-islands) - An integration that provides a set of utilities to work with Astro‚Äôs Server Islands.
