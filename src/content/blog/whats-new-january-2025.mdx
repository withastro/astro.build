### New Community-run Support Forum

Although we're proud of the welcoming, inclusive community environment we've created on Discord, not everyone in the Astroverse is active there. Some people aren't on Discord by choice. Some geographic regions make accessing Discord difficult. This can create issues getting the Astro support you need.

Our volunteer Support *Squad* may be in Discord, but the [Support *Squid*](https://get.supportsquid.ink/) is swimming in the open waters of the web! This is a brand new community initiative by some of our active Support Patrol members to help extend our support efforts and make questions and answers easier to find in web searches. ### Tips & Tools

The latest community-built utilities and integrations to help you build with Astro:


- [`@wix/astro`](https://www.npmjs.com/package/@wix/astro) - The Wix Astro integration provides session management for your Astro project.
- [Astro GitHub file loader](https://github.com/gingerchew/astro-github-file-loader) - Load files stored in a Github Repository into your Astro Content Layer.
- [Astro Integration Template](https://astro-integration-template.netlify.app/get-started/pick-a-template/) - Building an Astro integration? Just choose a template and get started!
- [`astro-ghostcms-loader`](https://github.com/MatthiesenXYZ/ghostcms-loader) -Use your GhostCMS content with the Astro Content Layer.

... and some built *with* Astro:

- [Party poll](https://astro-polls.pages.dev/) - Create your own poll! Built using PartyKit and Astro.
