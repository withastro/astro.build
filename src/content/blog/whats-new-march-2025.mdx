## Weird Wide Web

Some sites from the last month that show off what you can do with Astro that are quirky, innovative, inspiring or just plain fun!

- [SnowDayCal](https://snowdaycal.org/) - Will school be canceled tomorrow? Use this advanced snow day calculator to find out the probability based on current weather forecasts and historical snow day patterns.

- [MontFort](https://mont-fort.com/) - A stunning Astro site built with Three.js, GSAP and Storyblok CMS.
- [Interior Scene](https://vr-interior.oaksun.studio/) - A Matterport-style virtual tours demo with progressive texture loading in Three.js and Astro. Bonus: it has a banana FPS mode.

## Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. üõ†


- [`react-router-astro`](https://github.com/lilnasy/react-router-astro) - A starter template for creating a server-rendered SPA using `react-router` and Astro.
- [`astro-rerun-island`](https://github.com/TheOtterlord/astro-rerun-island) - An integration that reruns your persistent server islands on `astro:after-swap` to refresh your content without showing the user unnecessary loading screens.
- [`astro-meta`](https://github.com/ArifRabbaniIn/astro-meta) - Easily add meta tags to your Astro app.
- [`astro-ai-robots-text`](https://delucis.github.io/astro-ai-robots-txt/) - An Astro integration to disallow AI scrapers in `robots.txt`.
- [`s3-compatible-loader`](https://github.com/L4Ph/something-with-astro/tree/main/packages%2Fs3-compatible-loader) - A content loader to turn an S3-compatible object stroage into a CMS that works well with Obisidian's Remotly Save.
- [`astro-gemtext`](https://github.com/aspizu/astro-gemtext) - An Astro integration that lets you use `.gmi` (Gemini protocol document format) files as pages in your Astro site.
- [`astro-itchio`](https://artechfuz3d.github.io/astro-itchio-publisher/) - An Astro integration that automates your deployment to Itch.io.
- [Astro Clerk loader](https://github.com/LekoArts/astro-loaders/tree/main/packages/clerk) - Access a selection of Clerk Backend APIs by simply providing the method name.
- [Astro Plausible loader](https://github.com/LekoArts/astro-loaders/tree/main/packages/plausible) - Provides a Plausible content loader for Astro's content layer. You can access the Stats API v2 to view historical and real-time stats of your website.
- üåü [`starlight-markdown-blocks`](https://delucis.github.io/starlight-markdown-blocks/) - Extend Starlight‚Äôs Markdown asides syntax with custom (`:::block`) types.
- üåü [`starlight-toc-overview-customizer`](https://github.com/trueberryless-org/starlight-toc-overview-customizer) - Enhance Starlight's table of contents with customizable overview title
- [`reveal-animation-handler`](https://github.com/CarloBu/reveal-animation-handler) - A responsive text animation system for Astro that intelligently detects and respects natural line breaks. This utility provides dynamic, viewport-aware animations with multiple animation types and customizable options while maintaining responsive design across all devices.
- [`astro-appwrite-demo`](https://ark.sudovanilla.org/Korbs/astro-appwrite-demo) - A demo of using Appwrite on Astro, allowing end-users to create an account and update their username on the account page.
- [`astro-d1-magiclink`](https://github.com/OliverSpeir/astro-d1-magiclink) - Magic link auth based on Lucia-auth, Astro Actions, and Cloudflare D1.

... and some built *with* Astro:

- [WorksForMe](https://worksforme.online/) - Enter a time and a time zone(s) and get a shareable link so everyone around the world knows exactly when that meeting is!
- [Astro UI](https://github.com/BTS-CM/astro-ui) - A Bitshares UI built using Astro, React and Electron.
