## Latest releases

Minor releases introduce new features and improvements. Check out what's new!

- [Astro 5.6](https://github.com/withastro/astro/blob/refs/heads/main/packages/astro/CHANGELOG.md#560) - first-class astro:env and experimental session support in Cloudflare, more control over prefetching, and more!

- [Starlight 0.34](https://github.com/withastro/starlight/blob/main/packages/starlight/CHANGELOG.md#0340) - built-in heading anchor links, Tailwind 4 support, CSS cascade layers, and more!

### Open source feel-good Story

Last month's "What's New?" included a blog post by Angelika Cathor on [the implications of Astro's HTML streaming](https://angelika.me/2025/03/16/implications-of-astro-html-streaming/) and detailed their current workaround in use.

## Tips & Tools

The latest community-built utilities and integrations to help you build with Astro. üõ†

- [Astro Tanstack](https://github.com/universse/astro-tanstack) - Build a Single Page Application (SPA) with Astro, TanStack Router and Query.
- [cfetch](https://github.com/withstudiocms/cfetch) - An Astro integration that provides a cacheable fetch function for Astro SSR.
- [Astro WordPress](https://github.com/sijad/astro-wordpress) - Build WordPress themes using Astro.
- [Nova IDE Astro extension](https://extensions.panic.com/extensions/com.johnlindop/com.johnlindop.nova-astro/) - Provides syntax highlighting and basic autocomplete for `.astro` files for Panic's Nova IDE.
- üåü [Starlight Fullview Mode](https://windmillcode.github.io/starlight-fullview-mode/) - A minimalist plugin that collapses Starlight's sidebars and expands the main content to full width.
- [Inox Tools: cut short](https://inox-tools.fryuni.dev/cut-short) - An Astro integration that lets you stop processing a request instantly and send back a custom response, now with a `disableStreaming` option.
- [Typed Rest Routes](https://github.com/louisescher/typed-rest-routes) - An Astro integration that allows for creating type-safe server endpoints.
- üåü [Starlight scroll to top](https://frostybee.github.io/starlight-scroll-to-top/) - Add a scroll to top button to your documentation website.
- [Astro og](https://github.com/morinokami/astro-og) - Astro dev toolbar app that previews your page on social platforms and shows its open graph data.
- [Astro Google tag manager](https://www.npmjs.com/package/@picoo-studio/astro-google-tagmanager?activeTab=readme) - An integration for injecting Google Tag Manager snippets into Astro projects, supporting popular web analytics tools.

The latest community-built utilities and integrations to help you build with Astro. üõ†

... and some built *with* Astro:

- [Starlight Changelog](https://starlight-changelog.netlify.app/) - [Trueberryless](https://github.com/trueberryless/) (whose name, we learned this month, is *not* a pun on Drew Barrymore!) took inspiration from Astro's [Starlog theme](https://astro.build/themes/details/starlog/), added the [`astro-loader-github-releases` content loader](https://github.com/lin-stephanie/astro-loaders/tree/main/packages/astro-loader-github-releases) and gave us a pretty, dark-mode version of Starlight's own `CHANGELOG.md`, automatically updating. Bookmark it to keep up with Starlight's changes!
