## Releases

Astro's development keeps marching forward, with several notable releases in October:

- Astro 3.2: This update gives you more control over the browser history stack, enables page navigation with View Transitions, includes a route announcer for screen readers, and integrations can now dynamically add other integrations. Learn more in the [Astro 3.2 announcement](/blog/astro-320/).
- Astro 3.3: This release introduces an experimental `<Picture />` component, enhances syntax highlighting, brings package provenance, and adds various quality-of-life improvements. Learn more in the [Astro 3.3 announcement](/blog/astro-330/).
- Astro 3.4: Introducing Page Partials, along with improvements in image optimization and a first preview of an experimental dev overlay. Learn more in the [Astro 3.4 announcement](/blog/astro-340/).
- Starlight 0.11: Adds support for customizing Starlight‚Äôs UI with component overrides, adding attributes to sidebar links, and improves `<title>` tags. Learn more in the [Starlight 0.11 release notes](https://github.com/withastro/starlight/releases/tag/%40astrojs%2Fstarlight%400.11.0)

See the `#announcements` channel in Discord for all releases across the Astro repos, including Starlight, the Astro VS Code extension, compiler, packages, and integrations! Follow [Astro on GitHub](https://github.com/withastro/) for all updates.

## Docs

We extended the tutorial with some highly requested features!

- [Content Collections](https://docs.astro.build/en/tutorials/add-content-collections/)
- [View Transitions](https://docs.astro.build/en/tutorials/add-view-transitions/)

"Talking and Doc'ing" is our weekly live Discord call where we involve the community as we work on docs projects! Often we jump into a shared Codespace or HackMD document and everyone can participate. This month's activities included:

- Live reviewing and testing the two new mini-tutorials
- Improving our docs PR template and creating a new GitHub action to welcome first-time contributors
- All hands on deck reviewing translation submissions using the Astro Docs checklist to get everyone's `hacktoberfest-accepted` PRs merged
