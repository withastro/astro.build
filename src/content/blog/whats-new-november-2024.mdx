### Tips & Tools

The latest community-built utilities and integrations to help you build with Astro.

- [`astro-pandabox`](https://github.com/SaintSin/astro-pandabox) - A lightweight Lightbox and gallery component for Astro
- [Astro Embed: Baseline Status component](https://astro-embed.netlify.app/components/baseline-status/) - a component that displays browser support for a given web feature according to the Baseline project.
- [Astsro Embed: Bluesky post](https://astro-embed.netlify.app/components/bluesky/) - embed a Bluesky post on your Astro site
- [`astro-youtube`](https://www.npmjs.com/package/@barcia/astro-youtube) - a component to embed YouTube videos in your Astro site
- [Inox Tools: Astro 5 / Vite 6 support](https://inox-tools-git-feat-astro-5-fryuni-pro.vercel.app/) - all your favorite Inox Tools are ready for the latest version of Astro
- [`astro-typed-ids`](https://github.com/florian-lefebvre/astro-typed-ids) - get typed ids for your Content Layer entries, matching Astro 4 behavior.
- [Astro Bluesky post loader](https://github.com/ascorbic/astro-loaders/tree/main/packages/bluesky) - load and parse Bluesky posts, and use the data in your Astro site.
- [`bluesky-comments-tag`](https://github.com/ascorbic/bluesky-comments-tag)- A lightweight, customizable web component to use Bluesky for your website comments.
- [`starlight-contributor-list`](https://github.com/trueberryless-org/starlight-contributor-list) - display a list of all contributors to your GitHub project
- [`starlight-sidebar-topics-dropdown`](https://github.com/trueberryless-org/starlight-sidebar-topics-dropdown) - a Starlight plugin to add a drop down navigation to your sidebar
- [`starlight-cooler-credit`](https://github.com/trueberryless-org/starlight-cooler-credit) - Adds a Houston-themed credit to Starlight or Astro

... and some built *with* Astro

- [Simple Image Compressor](https://simple-image-compress.vercel.app/) - Upload an image, set your target file size, and download the compressed version.
- [1 checkbox per day](https://simplest-todo.pages.dev/) - The world's simplest to-do app! 
- [Astro Linky](https://astrolinky.netlify.app/) - Your Astro-powered URL Shortener & QR code generator.
- [Tract Stack](https://tractstack.com/) - a no-code website builder and content marketing platform.
