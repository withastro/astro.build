## Updates from Mission Control 

- 💼 [Astro Jobs](https://astro.jobs/) opens for business.
- 🦙 [TinaCMS goes Astro by default](https://tina.io/blog/astro-is-the-default-tinacms-starter), backed by community usage and demand.
- 🖼 [ImageKit releases an Astro integration](https://imagekit.io/docs/integration/astro) for real-time image & video resizing, automatic optimization, and file uploading.
- 🚀 [Astro 7 alpha preview](/blog/astro-620/#astro-v7-alpha) continues. Get an early jump on Vite 8 support, the stable Rust compiler, and more.

## Latest releases

Minor releases introduce new features and improvements. Check out what's new!

- [Astro 6.3](/blog/astro-630/) - Experimental advanced routing with Hono support, image redirect handling, and resilient island hydration.
- [Astro 6.4](/blog/astro-640/) - A new pluggable Markdown pipeline, a new Rust-based Markdown processor, and new Cloudflare helpers for advanced routing.
- [Starlight 0.39](/blog/starlight-039/) - More flexible autogenerated sidebars, improved styling, and stronger multilingual docs support.

The latest community-built [utilities and integrations](/integrations/) to help you build with Astro. 🛠


- [Astrobook](https://github.com/ocavue/astrobook): UI component playground that supports multiple frameworks, including React, Vue, Preact, Svelte, Solid, Lit, and Astro. It offers a unified environment to develop, test, and showcase components.
- [@hoangyell/gallery](https://gallery.hoangyell.com): Astro image gallery with lightbox, pyramid multi-row layout, and accessible controls.
- [@brandon_m_behring/book-scaffold-astro](https://github.com/brandon-behring/book-scaffold-astro): Astro 6 + MDX toolkit for long-form technical books.
- [@scale.digital/astro-bun](https://github.com/ScaliirDigital/root/tree/main/packages/astro-bun#readme): A native Bun adapter for Astro 6 — runs SSR, hybrid, and static sites directly on Bun.serve
- [astro-jsonfeed](https://github.com/myles/astro-jsonfeed): Generate a JSON Feed for your Astro site.
- [astro-oembed](https://github.com/hfournier/astro-oembed): Embed any URL in your Astro site using the oEmbed protocol.
- [eminence-astro-suite](https://eminence-astro-suite.xeffen25.com): Astro components and integration for managing page head, metadata, social sharing, icons, and SEO outputs.
- [astro-lang-i18n](https://github.com/TheElegantCoding/astro-lang-i18n#readme): 📦️ Smart multi-dimensional routing for Astro i18n projects.
- [astro-nice-asset-filenames](https://github.com/yaroslav/astro-nice-asset-filenames#readme): Astro integration for short, readable asset URLs.
- [astro-headers-file](https://github.com/hugo-vrijswijk/astro-headers-file): Astro integration that uses your existing configuration to write a static headers file for platforms like Cloudflare Pages, Netlify and Vercel, and others.
- [@dallay/astro-icon](https://github.com/dallay/astro-icon#readme): A community-maintained fork of `astro-icon` by Nate Moore, updated for Astro v6.x and Cloudflare Workers compatibility.
- [@develjet/astro-icon](https://github.com/develjet/astro-icon#readme): A lightweight Iconify integration for Astro with support for local SVG icons.
- [@erfianugrah/astro-image-hq](https://github.com/erfianugrah/astro-image-hq#readme): High-quality image service for Astro: encoder routing across avifenc, NVENC, and sharp with content-aware shadow detection.
- [@mannisto/astro-masonry](https://github.com/eremannisto/astro-masonry#readme): A masonry layout component for Astro.
- [@mgasme/elu](https://github.com/mateogasme/elu#readme): Composable Astro select component. Single, multi, searchable, grouped. Zero runtime dependencies.
- [@northsoon/phosphor-icons-astro](https://github.com/northsoon-studio/phosphor-icons-astro#readme): Phosphor icons as fully-typed Astro components — size, weight, color, mirrored and aria-label props out of the box.
- [@rafters/astro-data](https://github.com/rafters-studio/astro-data#readme): Loader and action contracts for Astro 6 sites.
- [@rgglez/astro-aside](https://www.npmjs.com/package/@rgglez/astro-aside): Astro aside/callout component with note, tip, caution, warning, and danger variants. Requires Tailwind CSS.
- [@rgglez/astro-downloadchecklist](https://www.npmjs.com/package/@rgglez/astro-downloadchecklist): Astro component that wraps a markdown task list and adds a CSV download button.
- [astro-bulma](https://github.com/lanesawyer/astro-bulma#readme): Astro component library wrapping Bulma CSS
- [astro-hotspot](https://github.com/whytspace/astro-hotspot#readme): Astro image service: hotspot cropping (x/y or '70% 34%') via Sharp's extract+resize.
- [astro-recommends](https://github.com/carvil/astro-recommends): Affiliate link cloaking for Astro. A static-site replacement for ThirstyAffiliates.
- [astro-site-shell](https://github.com/h-kono-it/astro-site-shell): A CLI-style terminal widget for Astro sites. Navigate content through a pseudo shell interface.
- [ezqr-astro](https://github.com/ezqr-ca/ezqr-astro): Astro component that embeds the EZQR free QR code generator on your site.


### CMS/text editors

- [WebcoreUI CMS](https://webcoreui.dev/cms): A CMS built for Astro with WebcoreUI. Includes a visual block builder, flexible content modeling, and a DB layer that just works.
- [@vulsecms/core](https://github.com/vulsecms/vulse#readme): Astro-native headless CMS for Cloudflare (D1, R2, Workers).
- [astro-blog-kit](https://www.npmjs.com/package/astro-blog-kit): A ready-to-use blog system for Astro with WordPress headless support, optional i18n, multiple layouts, and a comment system.

### Content loaders

- [notion-loader](https://github.com/ancs21/notion-loader#readme): Astro Content Loader for Notion as a CMS. Fork of `astro-notion/notion-astro-loader` with Astro 6 compatibility patches.

### Dev toolbar apps

- [astro-agent-annotate](https://github.com/robertvanhoesel/astro-agent-annotate#readme): Astro dev toolbar plugin: Alt+Click any element, annotate it, copy notes + source context to clipboard for AI agents.
- [astro-inspect-clip](https://github.com/XKonstX/astro-inspect-clip#readme): Astro dev-toolbar app to inspect elements and copy source info + instructions to clipboard.

### Astro integrations by and for other tools

- [@imagekit/astro](https://github.com/imagekit-developer/imagekit-astro#readme): Astro SDK for ImageKit.io - optimized image & video delivery with transformations
- [@a1local/astro-adpages](https://github.com/a1local/astro-adpages#readme): Astro integration helpers for AdPages local SEO and landing-page QA.
- [@dualmark/astro](https://github.com/dodopayments/dualmark/tree/main/packages/astro#readme): Astro integration for the Dualmark AEO framework.
- [@generative-dom/astro](https://github.com/generative-dom/generative-dom/tree/main/packages/adapters/astro): Official Astro integration for Generative DOM.
- [@hover-dev/astro](https://github.com/Hyperyond/Hover#readme): Astro integration for Hover. Injects the floating chat widget into your Astro dev site and starts the local agent-driving service.
- [@love-rox/tcy-astro](https://github.com/Love-Rox/tate-chu-yoko#readme): Astro integration and `<Tcy>` component for Japanese tate-chu-yoko (縦中横) auto-wrap
- [@pico-intl-dev/astro](https://github.com/Elioz404/pico-intl/blob/main/docs/guides/astro-middleware.md): Astro integration for pico-intl - SSR-ready, zero-overhead i18n for Astro projects
- [@sudodevstudio/fastly-for-astro](https://github.com/SudoDevStudio/fastly-for-astro#readme): Astro v6 adapter for Fastly Compute. Dynamic SSR, server islands, and KV-backed static assets with no app-code changes.
- [@unirate/astro](https://github.com/UniRate-API/astro-unirate#readme): Astro integration for UniRate — build-time currency exchange rate snapshots, `<Currency />` and `<Rate />` components, and runtime conversion helpers.
- [astro-resend-router](https://github.com/tmykkanen/astro-resend-router#readme): Astro integration for using Resend as an email-based mailing list router.
- [astro-typekit](https://github.com/eltrach/astro-typekit#readme): Astro-native font loading for Google Fonts and local font files.
- [astro-x402](https://github.com/morinokami/astro-x402#readme): Astro middleware integration for the x402 Payment Protocol.

### Markdown, Text, and more

- [@philnash/astro-related-content](https://github.com/philnash/astro-components/tree/main/packages/astro-related-content): An Astro integration that generates semantically related content for Markdown and MDX sources.
- [@casoon/astro-site-files](https://github.com/casoon/astro-site-files#readme): Astro integration that generates robots.txt, llms.txt, sitemap.xml, security.txt, and humans.txt at build time from typed configuration.
- [@jdevalk/astro-markdown-alternate](https://github.com/jdevalk/astro-markdown-alternate#readme): Astro integration that injects `<link rel="alternate" type="text/markdown">` into article pages at build time.
- [astro-mark-don](https://github.com/paultursuru/astro-mark-don#readme): Astro integration that generates a .md version of every page at build time, optimized for LLM crawlers.
- [astro-md-content-negotiation](https://www.npmjs.com/package/astro-md-content-negotiation): Astro integration that generates Markdown versions of every page at build time, enabling HTTP content negotiation.
- [astro-slop](https://github.com/yaroslav/astro-slop#readme): The most complete and flexible Astro integration for LLMs and generating markdown versions of pages. Per-page .md siblings, llms.txt index, content negotiation, and link rel=alternate.
- [@rafters/astro-meta](https://github.com/rafters-studio/astro-meta#readme): Build-time artifacts for crawlers and LLMs on Astro 6: JSON-LD, llms.txt, robots, sitemap, OG images, and build-time GEO readability scoring.
