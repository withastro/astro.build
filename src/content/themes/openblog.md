## Features:

- Minimal styling
- Mobile responsive
- 100/100 Lighthouse performance
- SEO-friendly with canonical URLs and OpenGraph data
- Sitemap support
- RSS Feed support
- Markdown & MDX support
- Syntax highlighting
- Image optimization
- Table of contents
- Dark mode
- Reading Time
- [Pagefind](https://pagefind.app/) static search library integration
- Related posts
- Share posts (Linkedin, twitter)
- Draft mode
- Copy code block
